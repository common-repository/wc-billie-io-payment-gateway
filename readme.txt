Use Billie.io on WooCommerce as Payment Gateway
 
== Description ==

This lightweight plugin allow you to use Billie.io in your WooCommerce Store.

<h4>Requirements</h4>
* <a href="https://de.wordpress.org/plugins/woocommerce/" target="blank">WooCommerce</a>
* <a href="https://www.billie.io/" target="blank">Billie.io</a> API Credentials

This plugin uses the official billie.io API.
The following requests are supported / used in this plugin.

<h4>Authentication</h4>
* Request OAuth Token
* Validate OAuth Token
<h4>Back-end Order Creation</h4>
* Create Order
* Ship Order

<strong>Important:</strong>
I am just a programmer who developed the plugin - i am not an employee of billie.io!
	
== Installation ==
 
1. Upload plugin over wordpress
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Activate the Gateway via WooCommerce -> Settings - > Payments - > Billie.io
